# JhfPhoto
仿照微信朋友圈上传照片的选择照片功能

#依赖的第三方库
https://github.com/nostra13/Android-Universal-Image-Loader<br>
https://github.com/chrisbanes/PhotoView

#简介
1.相册选择，照相机<br>
2.可以多选，设置最多选择的数量<br>
3.预览多选的照片（大图模式）<br>
4.大图展示照片（大图模式也可以选择）
